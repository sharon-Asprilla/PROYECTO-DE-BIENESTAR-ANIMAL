Consejos para los primeros auxilios en caso de emergencia

*Acércate con cuidado para evitar que el animal se asuste o huya.

*Observa si tiene heridas visibles, dificultad para respirar o signos de dolor.

*Si el animal está agresivo o muy asustado, usa una manta para cubrirlo antes de manipularlo.






Que hacer cuando hay hemorragias?

*Presión directa: Usa una gasa o paño limpio para detener el sangrado.

*Limpieza: Lava la herida con agua tibia y aplica un antiséptico sin alcohol.

*Vendaje: Cubre la herida con una gasa estéril y asegúrala con cinta médica.






Si hay fractura trata de hacer

*Inmovilización: Si sospechas una fractura, evita mover la extremidad afectada.

*Soporte: Usa una tablilla o periódico enrollado para estabilizar la zona.

*Urgencia: Lleva al animal a un veterinario lo antes posible.
